# AI-BOOK 书籍管理系统

一个基于 PHP 的在线书籍/章节管理系统。

## 功能特性

- 书籍管理
- 章节管理
- 用户登录认证
- 数据库管理

## 环境要求

- PHP 7.4+
- MySQL 5.7+
- Web 服务器 (Apache/Nginx)

## 安装步骤

1. 导入数据库文件
2. 配置 `config.php` 中的数据库连接信息
3. 访问首页进行安装

## 目录结构

```
├── api/            # API 接口
├── assets/         # 静态资源
├── includes/       # 公共文件
├── index.php       # 首页
├── login.php       # 登录页
├── novel.php       # 小说管理
├── chapter.php     # 章节管理
└── ...
```
